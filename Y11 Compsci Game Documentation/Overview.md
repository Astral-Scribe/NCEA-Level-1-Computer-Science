Indie Game Studio – Level 1 Year Brief  

## Context 

You are part of a small Indie Game Studio contracted to create a complete digital experience for a real audience. Over the year, your studio will design and develop: 

A playable game prototype (programming focus – Python as baseline). 

A public facing website that promotes and documents the game. 

A set of 3D assets that belong to the game world, suitable for digital display and optional 3D printing, AR and VR. 

Different parts of this work will be assessed against “Create a computer program” (92004) and “Develop a digital technologies outcome” (92005). 

## Game genres and scope 

Your studio must propose a game that fits into one or more of these beginner friendly genres: 

- Turn based strategy / tactics 
    

- Puzzle or tile-based logic 
    

- Arcade skill game (e.g., breakout, endless runner, simple shooter) 
    

- Narrative / choice driven adventure 
    

- Light tower defence / wave survival 
    

	- Simple management / clicker / resource game 
    

## The game must have: 

- A clear purpose (entertainment, learning, storytelling). 
    

- A specific target audience (age, interests, platform). 
    

- A well-defined core loop (what players repeatedly do). 
    

- Clear win/lose/progression conditions and a basic difficulty curve.
