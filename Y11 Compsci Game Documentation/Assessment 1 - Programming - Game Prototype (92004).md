## Brief 

Develop the core playable version of your Indie Game Studio title using Python (or another approved beginner friendly language). This is the “functional prototype” that proves your idea works as a game. 

# Requirements (92004 – programming) 

## Your program must: 

- Store and update multiple types of data using variables and at least one collection (e.g., list, array, dictionary). 
    

- Use selection and iteration (if/elif/else, loops) to control game logic, turns, or repeated actions. 
    

- Take user input (keyboard, mouse, or text commands) and produce meaningful output (onscreen changes, text, sound). 
    

- Be tested and debugged with evidence, including: 
    

- Planned test cases (normal, boundary, and invalid inputs). 
    

- Expected vs actual outcomes. 
    

- Debugging evidence (before/after code or descriptions of fixes). 
    

## Student documentation required: 

- A short game design summary (concept, audience, core loop, rules). 
    

- Programming decisions (data structures used, key algorithms). 
    

- Testing and debugging logs. 
    

- Reflection on how well the game meets its purpose and users’ needs
